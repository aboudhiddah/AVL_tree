фронтенд на vue, бекнд на python + flask 
для тестов pytest+selenuim
бд - neo4j

веб приложение, пользователь может авторизоваться под одной из трёх ролей:
1 - администратор - модерирует записи врачей, управляет данными филиалов
2 - врач - заполняет своё расписание, подтверждает и отменяет запись клиента. Может создавать медкарточку пациента, дополнять её
3 - клиент - выбирает филиал, дату, время и врача из базы, можно фильтровать по дате/времени или врачу. При записи должен указать вид и породу животного, его возраст и пол, а также описать проблему. Может отменить запись 

В рамках филиала предоставляется список услуг, которые предоставляются в этом филиале, предоставленные услуги врач заносит в карточку, администратор может сформировать отчет о проведенных услугах в одной клинике
